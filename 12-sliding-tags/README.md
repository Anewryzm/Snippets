# Sliding Tags — HML5 & CSS3 Snippet

**Demo**: [demo.webinterfacelab.com/12-sliding-tags](http://demo.webinterfacelab.com/12-sliding-tags/)

![Preview](http://www.webinterfacelab.com/assets/snippets/sliding-tags/preview.png)

This snippet is implemented with [Sass](https://github.com/nex3/sass), with help from [Bourbon](https://github.com/thoughtbot/bourbon).

To compile:

`sass --update scss:css -r ../bourbon/lib/bourbon.rb`

To compile and watch:

`sass --watch scss:css -r ../bourbon/lib/bourbon.rb`

## Browser Compatibility

* Firefox 4-11
* Safari 4-5
* Chrome 13-17
* Opera 10-11
* Internet Explorer 8-9
